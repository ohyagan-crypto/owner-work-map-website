# 藍星科技 tgbot 能力地圖

這是以主人目前 Telegram bot 的可公開資料整理出的網站，部署於 GitHub Pages：

https://ohyagan-crypto.github.io/owner-work-map-website/

## 本版重點

- 以 tgbot 的核心身份、回覆規則、技能流程與交付要求整理內容
- 呈現網站建置、圖片、影片、NotebookLM、監控、檔案交付、技能交接與查證能力
- 加入能力篩選互動
- 補上常用 Telegram 指令入口
- 補上安全邊界與近期任務案例

## 公開安全限制

公開網站不包含 token、API key、cookie、OAuth、密碼、瀏覽器資料、登入資訊、私人截圖、原始紀錄或工具輸出。

## 主要檔案

- `index.html`
- `styles.css`
- `app.js`
